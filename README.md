<h1 align="center">Roc8 - Dashboard Assignment</h1>

## Getting Started

- Clone the project

```bash
https://github.com/Avin008/roc8-user-dashboard-assignment
```

- Go to the project directory

```bash
  cd Tribble-A-Social-Media-App
```

- Install dependencies

```bash
  npm install / npm i
```

- Start the server

```bash
  npm run dev
```
